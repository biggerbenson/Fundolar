Official payment method logos
==============================

Place brand-approved artwork in this folder using these exact base names:

  stripe.svg       (or stripe.png)
  paypal.svg
  pesapal.svg
  flutterwave.svg
  paystack.svg

Recommended dimensions for the donation form tiles:
  - Display: ~56 x 36 pixels (width x height)
  - Source files: ~112 x 72 pixels (2x) for sharp rendering on high-DPI screens

Use SVG when the brand permits it. Do not ship trademarked artwork in a public
plugin repository unless your use complies with each brand’s media guidelines.

The bundled SVG files (if present) are neutral placeholders only; replace them
with official logos before production.
